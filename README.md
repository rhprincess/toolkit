## Installation

#### 1. Curl
`sh -c "$(curl -fsSL https://raw.githubusercontent.com/krinsofficial/toolkit/master/install.sh)"`

#### 2. Wget
`wget https://raw.githubusercontent.com/krinsofficial/toolkit/master/install.sh -O - | sh`

## Uninstallation

type `uninstall` in terminal
